## 검증 등급 (4단계)

항목별 실효성 검증은 tasks.json 상태가 아니라 이 등급으로 표기한다:

- 🔴 **코드만 존재** — 파일은 있으나 실행된 흔적(로그) 없음
- 🟡 **1회 실행 확인** — 실행 로그/출력 파일이 최소 1번은 있음
- 🟢 **실사용 검증** — 실제 상황에서 정상 동작한 증거(실제 승인/반려 요청이 오갔다, 실제 검색 결과가 나왔다 등)
- ⚫ **방치** — completed로 표시되어 있지만 이후 사용 흔적 없음 (일정 기간 후 재확인 필요)
