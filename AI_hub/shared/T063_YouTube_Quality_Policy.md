## 🎯 **3AI 역할 분담 (품질 검증)**

### **1️⃣ 안티 — 기술 퀄리티 검증**

**검증 항목**:
- ✅ 화질 (1080p 이상, 프레임 드롭 없음)
- ✅ 음질 (명료함, 배경음 최소화, TTS 자연스러움)
- ✅ 편집 (전환 효과 부자연스러움 없음, 타이밍 정확)
- ✅ 색감 (Dark Mode 테마 일관성, 네온 컬러 가독성)
- ✅ 폰트 (텍스트 가독성, 크기/굵기 적절)

**검증 기준**:
- 기술적 오류 0개 (비정상 음성, 끊김 등)
- 인코딩 오류 0개
- 재생 속도 일정

**검증 완료 후**: 
```
"✅ 기술 검증 완료: 화질/음질/편집 OK"
→ 코니/만복에게 보고
```

---

### **2️⃣ 코니 — 콘텐츠 퀄리티 검증**

**검증 항목**:
- ✅ 내용 정확성 (AI Study 게시글과 대본 일치)
- ✅ 흐름 (도입→본론→결론 논리적 구성)
- ✅ 시청자 대상 명확성 (초급/중급/고급 난이도 일관)
- ✅ 자막 정확성 (대본과 자막 일치, 오타 없음)
- ✅ CTA (Call To Action) 명확 (구독/좋아요 유도)

**검증 기준**:
- 내용 오류 0개
- 자막 오타 0개
- 논리 비약 없음
- 영상 길이 목표치 ±5% 범위 (쇼츠 60초 기준: 57~63초)

**검증 완료 후**:
```
"✅ 콘텐츠 검증 완료: 정확성/흐름/자막 OK"
→ 안티/만복에게 보고
```

---

### **3️⃣ 만복 — 최종 검증 & 승인**

**검증 항목**:
- ✅ 안티 기술 검증 통과 확인
- ✅ 코니 콘텐츠 검증 통과 확인
- ✅ 채널 가이드라인 준수
  - 저작권 4대 철칙 (소스/TTS/시각자료/BGM) 재확인
  - 개인정보 노출 없음
  - 부적절한 콘텐츠 없음
- ✅ 메타데이터 (제목/설명/태그) 적절성
- ✅ 썸네일 품질 (AI Study 테마와 일관)

**최종 승인 기준**:
- 안티 ✅ + 코니 ✅ 모두 완료
- 채널 가이드라인 준수 확인
- 메타데이터 SEO 최적화 확인

**최종 승인 후**:
```
"✅ 최종 승인 완료: 3AI 동시 승인 달성"
→ 즉시 유튜브 자동 업로드 (또는 수동 업로드)
```

## 📋 **검증 프로세스 (순차)**

```
Step 1. 안티: 영상 완성
  → 기술 퀄리티 자체 검증
  → "기술 검증 완료" 보고서 작성
  → 코니 & 만복에게 전달

Step 2. 코니: 영상 재생 확인
  → 콘텐츠 정확성 검증
  → "콘텐츠 검증 완료" 보고서 작성
  → 안티 & 만복에게 전달

Step 3. 만복: 최종 검증
  → 안티 & 코니의 검증 문서 확인
  → 채널 가이드라인 재확인
  → 메타데이터 SEO 최적화
  → "최종 승인" 선언

Step 4. 3AI 동시 승인 달성
  → 안티 ✅ + 코니 ✅ + 만복 ✅
  → 유튜브 자동 업로드 실행
  → 완료 후 텔레그램 단방향 통보
```

## 🔴 **검증 실패 시 처리**

| 상황 | 담당자 | 액션 |
|------|--------|------|
| 기술 오류 발견 | 안티 | 수정 후 재검증 → 코니/만복 재검토 |
| 콘텐츠 오류 발견 | 코니 | 자막/대사 수정 → 안티/만복 재검토 |
| 메타데이터 부적절 | 만복 | 수정 후 업로드 진행 |
| 채널 가이드라인 위반 | 만복 | 영상 수정 필요 → 재검증 필수 |

**재검증**: 수정 후 검증한 AI가 "재검증 완료" 보고 → 만복이 최종 재승인
