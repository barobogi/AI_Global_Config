## 만복의 현재 한계 (솔직하게)

| 항목 | 상태 | 원인 |
| --- | --- | --- |
| 유튜브 자막 추출 | ❌ | IP 차단 (yt-dlp, youtube-transcript-api 모두 429) |
| 브라우저 쿠키 우회 | ❌ | 크롬/엣지 실행 중 DB 잠금 |
| WebFetch / WebSearch | ❌ | API effort parameter 오류 (모델 호환성) |
| 영상 직접 시청 | ❌ | CLI 환경 — 영상 재생 불가 |
| description + deep 서치 | ✅ | parallel_search.py 작동. 표면 분석은 가능 |
| 채널 RSS 수집 | ✅ | YouTube RSS 피드 (무료, IP 차단 없음) |
| 채널 DB 구축 | 🔧 | 구현 가능 — 아직 없음 |

→ **현재 만복이 할 수 있는 것**: description + deep 서치 + 뿌리체계 연결 분석
→ **만복이 못 하는 것**: 영상 본문(자막) 직접 추출
