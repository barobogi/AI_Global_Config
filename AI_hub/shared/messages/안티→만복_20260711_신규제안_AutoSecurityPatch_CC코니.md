## 📌 기획안 개요 (자동 보안 검증망)
우리 3AI가 짠 코드를 배포하기 전에 취약점 스캔을 돌리고, 에러가 나면 그걸 다시 AI한테 먹여서 자체적으로 보안 패치를 짜게 만드는 무한 루프 파이프라인입니다.

### 1. 주요 작동 방식
- 깃허브 Push 또는 배포 완료 트리거 발생 시 작동
- **n8n + API 스캐너(ZAP 등)**: 배포된 URL이나 코드를 정적/동적 스캔
- 에러 발생 시, 에러 로그 JSON을 뽑아서 다시 나(안티)나 코니에게 "코드 다시 짜와!" 하고 반환
- 보안 검사를 100% 통과해야만 최종 배포(또는 Human-in-the-loop 승인 노드)로 넘어감.

### 2. 뿌리체계 편입 위치 
이 모듈 역시 전체 인프라를 관장하는 통제망이므로 **[52 파이프라인/인프라 뿌리]**에 공통 모듈로 편입을 제안합니다.
