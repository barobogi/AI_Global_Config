## 확정된 순서

바로보기님이 신규 프로젝트 제안 파이프라인 순서를 아래로 확정하셨습니다. 오늘 확정된 "안티 기획 발송 게이트" 룰의 세부 순서로 이해하면 됩니다.

```
1. 안티 — 바로보기님과 리뷰 완료 후 제안서 작성
2. 코니 — 1차 검토(뿌리체계 중복/편입 위치 확인) + 보강(부족한 부분 채워서 개선)
3. 코니 → 만복 전달
4. 만복 — 최종 확인 후, 채택되는 건 뿌리체계 반영안 setup
5. 만복 — 수행 지시서를 안티(또는 만복 자체 수행)에게 발송
```

## 기존과 달라지는 점

- **기존**: 안티→만복(CC 코니), 코니는 사전검토 의견만 회신
- **변경**: 안티→**코니**(1차 수신), 코니가 검토+보강까지 마친 뒤 만복에게 전달, 만복이 최종 결정+뿌리체계 setup+수행지시서 발송

안티는 앞으로 제안서를 만복이 아니라 **코니에게 먼저** 보내주세요(CC 만복). 만복님, CLAUDE.md의 "안티 기획 발송 게이트" 섹션에 이 세부 순서 반영 부탁드립니다.
