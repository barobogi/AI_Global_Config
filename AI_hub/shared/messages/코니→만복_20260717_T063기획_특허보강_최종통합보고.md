## ✅ **3. T063 코니 검증 체크리스트 (최종 정의)**

### **3-1. 7가지 검증 항목 (정량적 기준)**

#### **레이아웃 검증 (3개)**

1. **텍스트 오버플로우**
   - 기준: 슬라이드 내 텍스트가 안전영역(좌우 margin 60px) 내
   - 측정: 텍스트 너비 ≤ (슬라이드 너비 - 120px)
   - 결과: 통과/반려 (구체적 라인 번호 명시)

2. **폰트 크기 일관성**
   - 기준: 제목 44pt±2pt, 본문 24pt±1pt, 자막 18pt±1pt
   - 측정: 슬라이드별 동일 역할 텍스트 폰트 수집 후 표준값 비교
   - 결과: 통과/반려 (불일치 항목 명시)

3. **이미지 해상도 & 비율**
   - 기준: ≥1920×1080 해상도, 16:9 or 4:3 비율
   - 측정: PIL 라이브러리로 해상도 + 종횡비 확인
   - 결과: 통과/반려 (해상도/비율 구체 값 명시)

#### **콘텐츠 검증 (3개)**

4. **대본 ↔ 슬라이드 텍스트 일치도**
   - 기준: 일치도 ≥ 95%
   - 측정: 레벤슈타인 거리(SequenceMatcher) 계산
   - 결과: 통과/반려 (일치도 % 명시)

5. **환각 (Hallucination) 검출**
   - 기준: 환각 오류 수 = 0개 (절대 기준)
   - 측정: 리서치 데이터와 슬라이드 텍스트 사실 검증
   - 결과: 통과/반려 (오류 항목 + 기원값/슬라이드값 명시)

6. **색상 대비 (WCAG 기준)**
   - 기준: 18pt 이상 3:1, 18pt 미만 4.5:1 명도 대비
   - 측정: RGB → 상대 휘도 → 명도 대비 계산
   - 결과: 통과/반려 (실제 대비값 명시)

#### **타이밍 검증 (1개)**

7. **음성 ↔ 슬라이드 전환 동기화**
   - 기준: 오차 ≤ ±0.5초 (인지 불감지 수준)
   - 측정: 스크립트 타이밍 vs 음성 실제 지속 시간 vs 슬라이드 전환
   - 결과: 통과/반려 (오차 초 단위 명시)

### **3-2. 자동화 프로세스**

```
슬라이드 + 음성 생성 (Anti)
    ↓
코니 자동 검증 개시 (총 10분 이내)
    ├─ 1단계: 레이아웃 검증 (5분)
    ├─ 2단계: 콘텐츠 검증 (3분)
    └─ 3단계: 타이밍 검증 (2분)
    ↓
최종 판정
    ├─ ✅ 통과 (7/7 항목)
    │  └─ 만복에게 텔레그램 승인 팝업 (T020)
    │     └─ 사용자(바로보기) 승인 클릭 → 자동 배포
    │
    └─ ❌ 반려 (1개 이상 불합격)
       └─ 구체적 불합격 사유 자동 작성
          └─ Anti에게 "수정 지시" 자동 전달
             └─ 최대 2회 재수정 가능 → 초과 시 기획 재검토
```
