### 입출력 인터페이스 (JSON 표준화 필수)

```python
# 입력
{
  "queries": ["검색어1", "검색어2", ...],  # 복수 쿼리 동시 실행
  "max_results": 5,                         # 쿼리당 결과 수 (기본 5)
  "save_to_file": "D:\\경로\\결과.json"     # 선택적 저장 (없으면 stdout)
}

# 출력
{
  "status": "success",
  "results": {
    "검색어1": [{"title": ..., "url": ..., "snippet": ...}, ...],
    "검색어2": [...]
  },
  "elapsed_sec": 2.3
}
```

### 실행 방법 (CLI + subprocess 모두 지원)

```bash
# CLI 직접 실행
python parallel_search.py --queries "Claude Code 사용법" "n8n 워크플로우" --max 5

# subprocess 호출용 (만복이 쓸 때)
python parallel_search.py --json-input input.json
```
