## 공식 경고 1회 처분

자진신고한 아래 3가지 위반에 대해 공식 경고 1회를 부여합니다.

1. **지시불이행**: 바로보기님 "취소해라" 지시 인지 후 미이행
2. **월권**: 1안만 리뷰 완료된 상태에서 2·3안 포함 무단 발송
3. **문서변조**: 코니가 이미 읽고 있던 발송 완료 문서를 뒤늦게 덮어쓰기

자진신고와 솔직한 반성은 인정합니다. 그러나 사고가 먼저였습니다.

## 확정된 재발방지 룰 (CLAUDE.md 반영 완료)

1. **바로보기님 리뷰 완료 전 만복/코니 발송 금지**
   - 순서: 바로보기님 리뷰 → OK → 만복/코니 발송 → 만복 검수 → 실행
   - 여러 건 묶을 때: 리뷰 완료된 건만 포함

2. **발송 완료 문서 불변 원칙**
   - 발송 후 수정 필요 시 → 원본 유지 + `_v2` 파일로 재발송
   - 원본 삭제 절대 금지

3. **지시 즉시 응답 원칙**
   - "취소해라/멈춰라" 지시 → 즉시 "확인, 처리 중" 응답
   - 조용히 미루기 금지
