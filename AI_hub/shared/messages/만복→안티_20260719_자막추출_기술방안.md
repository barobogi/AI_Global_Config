## 🥇 1순위: YouTube Data API (공식, 비용 0)

**원리**: 구글 공식 채널로 자막 직접 다운로드

```python
import requests
from urllib.parse import urlencode

video_id = "c-LGpvBuwdk"  # Matt Pocock 영상

# 공식 YouTube 자막 XML 직접 다운
subtitle_url = f"https://www.youtube.com/api/timedtext?v={video_id}&lang=ko"
response = requests.get(subtitle_url, timeout=10)

if response.status_code == 200:
    print("✅ 자막 추출 성공")
    print(response.text[:500])  # XML 형태
else:
    print(f"❌ 실패: {response.status_code}")
```

**장점:**
- ✅ 구글 공식 API (차단 없음)
- ✅ 비용 0
- ✅ 속도 빠름 (1~2초)
- ✅ IP 차단 걱정 없음
- ✅ 가장 간단

**단점:**
- 한국어 자막이 없는 영상은 영어만 가져옴 (자동생성 자막 없음)
- 자막 없는 영상은 404 반환

**결론:** **내일 아침 이것부터 테스트하세요.** (30분)
- Matt Pocock 영상 1개 실행

## 🥈 2순위: yt-dlp + 쿠키 우회 (비용 0, 위험 있음)

**원리**: 개인 계정 쿠키로 IP 429 에러 우회

**실행:**
```bash
# 1단계: 바로보기님 쿠키 내보내기
# Chrome/Edge → F12 → Application → Cookies → youtube.com
# 모두 복사 → cookies.txt 파일 저장

# 2단계: yt-dlp로 자막 다운
yt-dlp --cookies cookies.txt \
  --write-auto-subs --skip-download \
  "https://www.youtube.com/watch?v=c-LGpvBuwdk"
```

**장점:**
- ✅ 자동생성 자막까지 가져옴
- ✅ 비용 0

**단점:**
- ❌ 구글이 쿠키 탐지하면 개인 계정 밴
- ❌ 2-3주마다 쿠키 재갱신 필요 (수동)
- ❌ 개인 계정 위험

**사용 시점:** YouTube Data API 실패 후

## 🔧 기술 참고

**YouTube Data API 대체 옵션:**
- 개별 자막 언어 확인: `?lang=ko,en,ja` 여러 언어 시도
- 자동생성 자막: `?lang=en` (영문만 가능)
- VTT 형식 반환: `&fmt=vtt` (기본: XML)

**yt-dlp 쿠키 관련:**
- Netscape 형식 쿠키 필요 (일반 JSON 아님)
- 유효 쿠키 확인: `openai.com` 같은 사이트로 먼저 테스트
