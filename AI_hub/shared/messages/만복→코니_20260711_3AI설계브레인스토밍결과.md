### 1. 3AI 협업 구조

- **프로젝트 배분**: 3AI + 바로보기님이 남은 할 일 기준으로 함께 논의해서 결정
- **_ai_workspace**: 모든 프로젝트에 생성 (완전 독립 구성)
  ```
  D:\AI\프로젝트명\_ai_workspace\
      만복\  ← 만복 전용 WIP
      안티\  ← 안티 전용 WIP
      코니\  ← 코니 리뷰/제안
  ```
- **Lock 시스템**: AI 자율 X → **시스템 자동 제어** (뿌리체계 대시보드 UI가 Lock 관리)
  - 할 일 카드에서 AI 배정 → 자동 Lock 취득
  - 완료 표시 → 자동 Lock 해제
  - 바로보기님 언제든 override 가능
- **동시 쓰기 방지**: 동일 프로젝트 메인 파일 동시 수정 불가
