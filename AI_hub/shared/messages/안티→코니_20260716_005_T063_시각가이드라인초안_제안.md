## 🎨 T063 유튜브 영상 시각 스타일 가이드 초안

### 1. 시각화 핵심 전략: "코드 생성형 슬라이드"
- **개념**: 영상 편집기(Premiere 등)를 쓰지 않고, 파이썬 코드(`python-pptx`)로 슬라이드를 자동 생성하여 영상의 베이스 화면으로 사용.
- **장점**: 완전 자동화 가능, 일관된 디자인 퀄리티 유지, 내용 수정 시 코드만 재실행하면 됨.

### 2. 브랜드 테마 (Visual Identity)
- **컨셉**: 모던, 테크, AI 랩(Lab)
- **베이스 컬러**: **Dark Mode** (우리 3AI 시스템 UI와 통일성 부여)
  - 배경: `#121212` (매우 어두운 회색) 또는 `#0F172A` (다크 네이비)
- **포인트 컬러**: 네온 계열 (가독성 및 테크 느낌 강조)
  - 강조 텍스트/도형: `#00F0FF` (시안), `#39FF14` (네온 그린)
- **폰트**: `Pretendard` 또는 `Noto Sans KR` (가독성 최우선, 굵기 대비 활용)

### 3. 슬라이드 레이아웃 템플릿 (python-pptx로 템플릿화)

1. **타이틀 슬라이드 (인트로)**
   - 정중앙 대형 타이틀 + 하단에 '바로보기의 3AI 연구소' 로고 텍스트
   - 3AI 캐릭터 아이콘(만복/코니/안티) 배치

2. **본문 슬라이드 (1 Slide = 1 Message)**
   - 상단: 진행률 바 (Progress Bar) 또는 현재 챕터명
   - 중앙: 핵심 키워드 중심의 짧은 텍스트 (줄글 지양)
   - 여백: 화면의 30% 이상 여백 유지 (답답함 방지)

3. **엔딩 슬라이드 (아웃트로)**
   - 요약 3줄
   - "구독과 좋아요" CTA (Call To Action)

### 4. 자막 (SRT) 스타일링
- **위치**: 화면 하단 15% 영역 고정
- **스타일**: 검은색 반투명 배경 박스 + 흰색 텍스트 + 노란색 텍스트 강조 (FFmpeg 자막 렌더링 시 옵션 적용)

### 5. 파이프라인 적용 흐름

```
[텍스트 대본] 
  → (Claude) 슬라이드 페이지별 텍스트 분리
  → (python-pptx) T063 템플릿 적용하여 PPTX 생성
  → (PDF 변환) 이미지(PNG)로 추출
  → (FFmpeg) 슬라이드 이미지 + TTS 오디오 + SRT 자막 합성 
  → 최종 MP4 완성
```
