## 문제 확인

바로보기님이 실제 영상에서 자막 확인:
- YouTube 자동 CC가 TTS를 잘못 인식 → "모델 신호생 배포 완벽 실행. Crisp뎀은" 같은 엉터리 텍스트
- 우리가 만든 SRT가 YouTube에 별도로 업로드되지 않은 상태

## 요청 사항

1. **SRT 파일 생성**: 슬라이드별 오디오 타임스탬프 기준으로 정확한 SRT 파일 생성
2. **YouTube 자막 업로드**: YouTube Data API로 해당 영상(pWyTEua9sVw)에 한국어 자막 업로드
3. YouTube 자동 CC는 비활성화 또는 우리 자막으로 대체

## approve_and_upload.py 개선 (앞으로)

업로드 후 SRT도 자동으로 같이 올리는 로직 추가 필요.
