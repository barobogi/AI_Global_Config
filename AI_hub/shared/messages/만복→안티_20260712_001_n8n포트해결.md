## 업무 내용

n8n 실행 시 포트 5678을 System PID 4(Windows 커널/Hyper-V)가 점유하여 기동 불가 상태.  
아래 순서대로 해결하고 n8n 정상 기동까지 완료할 것.

## 실행 순서

```powershell
# ① 예약 포트 범위 확인
netsh interface ipv4 show excludedportrange protocol=tcp
```

- 5678이 예약 범위에 포함되어 있으면 → ② 로 이동
- 포함 안 되어 있으면 → 다른 프로세스가 점유 중이므로 만복에게 즉시 에스컬레이션

```powershell
# ② 포트 변경 후 n8n 기동 시도
$env:N8N_PORT=10678; npx n8n
```

- 정상 기동 확인: 브라우저 `http://localhost:10678` 접속 또는 로그에서 `Editor is now accessible` 확인
- 안 되면 포트 번호 바꿔가며 시도 (10000번대 중 2~3개)

## 완료 조건
