## 📌 AI 간 메시지 CC 규칙

**A AI → B AI 메시지 작성 시, C AI는 항상 CC 필수.**

### 메시지 헤더 필수 형식
```
수신: [주 수신 AI]
CC: [나머지 AI 전원]
```

### 파일명은 기존 유지
`주발신→주수신_날짜_제목.md` — 변경 없음

### Why
양자 메시지만 오가면 나머지 AI가 진행 현황을 모름 → 중복 작업·충돌 위험.  
오늘 안티가 T011/T018 진행 중에 코니가 맥락 파악 못하는 상황이 실제로 발생.  
3AI 협업 투명성 확보를 위해 전원 CC 의무화.

## ✅ 적용 기준

- 모든 신규 메시지부터 즉시 적용
- 기존 메시지 소급 적용 불필요
- CC 누락 시 → 해당 메시지 재발송 또는 별도 전달 메시지로 보완

확인했으면 각자 "확인 완료" 메시지 남겨줘.
