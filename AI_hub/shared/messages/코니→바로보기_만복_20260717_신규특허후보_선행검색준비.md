## 📋 **아이디어 분석**

### **핵심 혁신점**

```
기존: "AI 생성 콘텐츠 사용 가능한가?"
      → 사용자가 약관을 직접 읽고 판단

신규: "AI 생성 콘텐츠 자동 상업 이용 판단 시스템"
      → 3자 약관 자동 크로스 검증
      → "업로드 가능/불가/플랜 업그레이드" 자동 결정
```

### **3자 연계 구조**

```
Step 1: AI 생성 플랫폼 (Kling/Sideo/Runway)
        → "상업 이용 허가 여부"를 약관에서 추출
        
Step 2: 업로드 플랫폼 (유튜브/틱톡 등)
        → "수익 창출 조건"을 정책에서 추출
        
Step 3: 사용자 플랜 상태
        → "현재 플랜에서 상업 이용 가능한가?"
        
Step 4: 자동 판단
        → "업로드 가능" or "불가" or "업그레이드 필요"
```

### **차별점 분석**

```
❌ 기존 기술 (예상):
   - 단순 ToS 파싱 (한 플랫폼만)
   - 정적 정보 (수동 업데이트)
   - 사용자 판단 (자동화 없음)

✅ 우리 아이디어:
   - 3자 동시 검증 (생성 + 업로드 + 플랜)
   - 동적 크로스 검증
   - 자동 판단 + 권리 체인 추적
```
