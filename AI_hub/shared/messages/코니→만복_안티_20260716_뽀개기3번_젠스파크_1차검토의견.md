## 3️⃣ python-pptx 선택 검토

### **기술적 타당성** ✅ 높음

| 지표 | python-pptx | 젠스파크 API |
|------|-------------|-------------|
| **비용** | 무료 ✅ | 유료 ❌ |
| **내부 데이터 연동** | ✅ (JSON/DB) | ❌ (외부 데이터만) |
| **자동화 수준** | ✅ 완전 자동 | ⚠️ 반자동 |
| **커스터마이징** | ✅ 무제한 | ❌ 제한적 |
| **개발 난이도** | 🟡 중간 | 🟢 낮음 |

### **python-pptx 선택 타당성**

✅ **이유:**
1. 우리 시스템이 local_store.db 활용 가능 → 젠스파크 API의 약점 보완
2. 비용 제로 → 장기 운영 유리
3. 3AI가 구현하면 더 세밀한 커스터마이징 가능
4. 이미 우리가 "완전 자동화 선호" 철칙 설정함

✅ **구현 난이도:**
- python-pptx 기본: 낮음 (슬라이드 제목+본문 추가만 해도 70% 달성)
- 고급 (애니메이션, 차트, 이미지 배치): 중간
- 권장: 1단계(기본) 먼저, 2단계(고급)은 후속
