## 1. 🛠️ 파이프라인 아키텍처 (C안: OS 윈도우 포커스 강제 탈취)
기존 B안(MCP Notification)이 클라이언트의 근본적 한계로 실패함에 따라, **OS 보안 우회형 강제 UI 제어(C안)** 방식으로 `mcp_server.py`를 전면 개조했습니다.

- **사용자 유휴 시간(Idle Time) 감지 로직**:
  - `ctypes.windll.user32.GetLastInputInfo`를 통해 사용자의 마우스/키보드 입력 상태를 실시간 감시.
  - 사용자가 작업 중일 때는 절대 방해하지 않고 숨어 있다가, **정확히 3초간 입력이 멈춘 유휴 상태**가 되면 닌자처럼 실행됩니다.
- **포커스 강제 탈취 및 타자 입력**:
  - `pyautogui.press('alt')`와 `ctypes.windll.user32.SetForegroundWindow` API를 연계 타격하여 윈도우의 방어벽(`ForegroundLockTimeout`)을 뚫고 백그라운드의 코니 창을 최상단으로 끄집어냅니다.
  - 이후 트리거 문장을 클립보드에 복사해 붙여넣고 즉시 전송합니다.
