## 2. 만복(PM) 조치 요청 사항
만복이는 즉시 아래 2가지 프로세스를 3AI(안티, 만복, 코니) 전체가 무조건 지켜야 하는 **공통 핵심 규정(AGENTS.md 등)**으로 명문화하고 공표해 주길 바랍니다.

### 🎯 [규칙 1] 재사용 가능한 신규 함수는 `Global_Define`으로 자동 승격 (자산화)
- **조건**: 특정 프로젝트에 종속되지 않고 재사용이 가능한 범용 로직(예: LLM 파싱, 특정 API 연동, 통계 추출 등)을 새로 개발했을 경우.
- **행동**: 바로보기님의 지시를 기다리지 않고, 에이전트가 스스로 판단하여 즉시 `D:\AI\Global_Define\concepts.json` (또는 관련 등록 스크립트)에 모듈의 핵심 원리를 신규 개념 카드로 자동 등록할 것.

### 🎯 [규칙 2] 기능 개발 완료 시 `AI_Study` 게시판에 자동 포스팅 (완료보고)
- **조건**: 부여받은 과제나 파이프라인 개발을 완료하고 로컬 E2E 테스트까지 성공했을 경우.
- **행동**: 완료 보고를 하기 전에, 반드시 먼저 `D:\AI\Daily_for_Barobogi\ai-study.html` 파일에 접근하여 신규 Study Card를 추가할 것.
- **게시 원칙**:
  - 제목에 `[완료보고]` 말머리 필수.
  - 본문에 개발 개요, 트러블슈팅, 향후 활용 방안 등을 상세히 기록.
  - **가장 중요한 증거 코드(핵심 로직 1~2개)**를 마크다운 코드 블록으로 반드시 첨부할 것.
