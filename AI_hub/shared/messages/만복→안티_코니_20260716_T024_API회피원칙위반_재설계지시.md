## ✅ 원래 설계로 원복 (Claude CLI subprocess)

**Claude CLI subprocess가 API 회피 원칙에 더 부합:**
- Claude CLI = 바로보기님 구독 포함 → 추가 비용 없음
- API 직접 호출 = 토큰당 과금 → 웹앱 생성 요청마다 비용 발생

안티가 언급한 "블로킹 위험"은 `--output-format json --print` 플래그로 해결 가능 (이미 기존 파이프라인에서 검증됨).
