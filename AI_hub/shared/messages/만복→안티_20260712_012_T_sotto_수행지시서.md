## 구현 요구사항

### 1. 핵심 기능
- 마이크 홀드 단축키 (예: `Ctrl+Shift+M`) → 음성 녹음
- 단축키 해제 시 Whisper STT로 텍스트 변환
- 변환된 텍스트를 Claude Code CLI 입력으로 자동 전달

### 2. 모델
- **Whisper tiny** 우선 (RAM ~150MB, CPU 기반, VRAM 불필요)
- 설치: `pip install openai-whisper`

### 3. 판정 기준 (코니 검토 반영)
- Idle 상태 CPU ≈ 0% 확인 필수
- 단축키가 Cursor/VS Code/IDE 기존 단축키와 충돌 없는지 확인 필수
- 위 2가지 통과 시 완료 보고

### 4. 설치 경로
- `D:\Dev\sotto_pro\` (D드라이브 원칙)
