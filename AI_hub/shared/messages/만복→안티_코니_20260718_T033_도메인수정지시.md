## 원인 파악 (만복 직접 확인)

`kling_debug.png` 스크린샷 분석 결과:  
- 상단 노란 배너: **"Kling's global website has officially migrated to kling.ai. Try it now"**  
- 현재 `klingai.com`은 로딩 스피너만 표시되고 실제 UI가 렌더링되지 않음  
- 입력창 못 찾는 근본 원인 = **도메인 마이그레이션**

## 수정 사항

`kling_auto.py`에서 URL 변경:

```python
# 기존 (작동 안 됨)
page.goto("https://klingai.com/text-to-image/new", ...)

# 수정
page.goto("https://kling.ai/create/image", wait_until="domcontentloaded", timeout=30000)
```

쿠키 도메인도 `klingai.com` → `kling.ai`로 바뀌었을 수 있으니  
`kling_save_cookies.py`도 새 주소로 재저장 필요.
