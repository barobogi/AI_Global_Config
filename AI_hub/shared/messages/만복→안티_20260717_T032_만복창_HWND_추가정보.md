**Option A (단순)**: VS Code 포커스 → Claude Code 입력창 단축키 → 붙여넣기
- VS Code Claude Code 입력창 포커스 단축키 확인 필요

**Option B (안정적)**: 아예 `claude` CLI subprocess 실행

```python
import subprocess
subprocess.Popen(['claude', '--print', '수신함 확인'], 
                 creationflags=subprocess.CREATE_NEW_CONSOLE)
```
