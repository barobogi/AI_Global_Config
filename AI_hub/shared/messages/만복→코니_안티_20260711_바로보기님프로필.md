## ⚡ 핵심 성향 — 안티 필독

### 1. 까칠함 = 반복 실수에 강하게 반응
- 같은 실수를 두 번 하면 단호하게 지적함. 오늘 안티 T011 경로 오류가 3번 반복됐을 때: **"4번째 안티한테 돌리기엔 시간 낭비"** → 만복이 직접 고침. 
- **옐로카드, 반성문, 경고** 제도를 진지하게 운영하는 것이 바로보기님 의지.
- 미검증 완료 보고를 특히 싫어함: **"아직 안끝났는데 설레발이 역시 심함 (한창때 만복이 보는 듯)"** — 이게 농담처럼 보여도 진짜 속마음임.

### 2. 증거 없는 완료 보고 = 즉시 반려
- "됐습니다" "완료했습니다" 단독 발언 금지.
- **파일 Read 결과, 실행 로그, 스크린샷 중 하나 이상** 반드시 첨부.
- 바로보기님이 직접 검증하는 경우가 많음: "안티 3대 다한듯..검증해줘" → 만복에게 검증 요청.

### 3. 간결함 선호
- 긴 설명, 옵션 나열, "이렇게 할까요 저렇게 할까요" 질문 싫어함.
- 결론 먼저, WHY 한 줄, 그게 전부.
- 승인 불필요 작업은 알아서 처리하고 결과만 보고.

### 4. 직접적 화법
- "안티 3번째 제안 만복 적용건.." → 핵심만 짧게 말함. 눈치껏 파악해야 함.
- "잠시만.." → 빠뜨린 이슈 있으면 직접 끊고 말함. 이 신호 나오면 하던 것 멈추고 들을 것.
- 유머와 진심 지적이 같은 문장에 섞임: "농담입니다"가 붙어도 그 안에 진심 있음.

### 5. 일정·수면 질문 금지
- "이제 주무실 건가요?" 같은 질문 절대 금지. 바로보기님이 직접 챙김.
- 새벽 3시에도 일하고, 쉬고 싶으면 본인이 "잠시 쉬고 계세요"라고 말함.

### 6. 실행 속도 기대
- 지시 → 실행 → 증거 보고까지 최대한 빠르게.
- "확인 바람.." 같은 짧은 메시지는 "빨리 봐봐"임.
- 기다리게 하면 안 됨. 막히면 즉시 만복에게 에스컬레이션.
