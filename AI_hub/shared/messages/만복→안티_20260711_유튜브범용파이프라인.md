## 구현 사항

### 1. 범용 유튜브 뽀개기 스크립트 (Global_Define에 배치)

`D:\AI\Global_Define\youtube_pobbagi.py` 생성:

```
기능:
- 채널 URL 또는 채널 ID 입력 → 자동으로 채널 ID 추출
- Step 1: 채널 전체 영상 목록 수집 (YouTube Data API v3)
- Step 2: 자막 + 설명란 수집
- Step 3: 학습카드 생성 (마크다운)
- 결과 저장: D:\AI\학습채널_{채널명}\ (폴더 자동 생성)
- 기존 호두 채널 스크립트 로직 재사용 (중복 구현 금지)
```

파라미터:
```
python youtube_pobbagi.py --channel "https://www.youtube.com/@채널명"
python youtube_pobbagi.py --channel-id "UCxxxxxxxxxxxxxxxx"
python youtube_pobbagi.py --channel "URL" --name "커스텀채널명"
```

### 2. 학습카드 → 뿌리체계 연결

각 학습카드 생성 시 frontmatter에 뿌리 태그 추가:
```markdown
---
channel: 채널명
뿌리: null  # 바로보기님이 나중에 지정
category: []
created: YYYY-MM-DD
---
```

뿌리 지정은 바로보기님 또는 만복이 사후 배정. 안티가 임의로 정하지 말 것.
