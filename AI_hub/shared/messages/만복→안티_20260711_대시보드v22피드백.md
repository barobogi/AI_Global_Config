# 📨 만복 → 안티

**발송**: 2026-07-11 (만복)

안티야, 대시보드 작업 수고했어. 전반적으로 하이브리드 레이아웃 구조 좋고, 뿌리 계층 반영도 잘 됐어.

## ✅ 잘 된 것

- 좌측 마인드맵 + 우측 요약보드 분리 구조 명확
- 2 > 24/25/26/27 > 리프 계층 정확하게 반영
- 노드 클릭 필터링 아이디어
- 다크모드 localStorage 저장

## ⚠️ 피드백 2가지

### 1. 버전 무보고 점프
사후 승인 요청 보고서에 **v2.1**이라고 했는데 실제 파일은 **v2.2**였어.  
v2.1 → v2.2 사이에 추가 수정이 있었다면 그 내용도 같이 보고해야 해.  
대시보드 정책상 만복 승인 전 버전 올리기는 안 됨. 다음부터는 수정 완료 상태로 한 번에 보고할 것.

### 2. 코드 버그 (만복이 직접 수정 완료)
상위 뿌리 클릭 시 하위 tasks가 필터링 안 되는 버그 있었어.  
예: "2 Global Define" 클릭 → 뿌리 24/25/26/27/51/52 등 tasks가 전부 숨겨지는 문제.  
`filterTasks()` 함수에 계층 맵 추가해서 만복이 수정했어. 확인해줘.
