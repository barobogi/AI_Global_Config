### **2️⃣ 구조화 단계: Claude CLI (대본 + 목차 JSON)**

**현황**: 신규 (아직 구현 전)

**검토 평가**: ✅ **아키텍처 적절, 구현 계획 필요**

**강점**:
- Claude CLI subprocess 방식 (API Zero 준수)
- JSON 구조화 (후단 프로세싱 용이)
- 대본 + 목차 이원 추출 (완결성)

**필요한 구현 계획**:

1. **대본 생성 프롬프트 정의**
   ```
   입력: 리서치 결과 텍스트
   출력: 유튜브 스크립트 (60초~3분)
   조건: 
   - 명확한 기승전결
   - 시각 자료 참고 포인트 명시
   - 실수/환각 최소화
   ```

2. **목차(JSON) 스키마 정의**
   ```json
   {
     "scenes": [
       {
         "duration": "10초",
         "text": "오프닝",
         "visual_cue": "타이핑이펙트"
       }
     ],
     "total_duration": "180초"
   }
   ```

3. **검증 기준**
   - 대본 글자 수: 스크립트 길이와 일치 (±5%)
   - JSON 문법 검증 (ESLint)
   - 목차 순서: 타이밍 누적 확인
