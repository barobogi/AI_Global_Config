### 1. ADSP 4과목 핵심 개념

| 과목 | 핵심 내용 |
|------|-----------|
| ① 데이터 이해 | 빅데이터 3V (Volume/Velocity/Variety), 정형/비정형 분류 |
| ② 분석 기획 | CRISP-DM 프로세스, KDD 방법론, ROI 기반 우선순위 |
| ③ 데이터 분석 | 통계 기초, 회귀/분류/군집, 시계열 분석 |
| ④ 데이터 시각화 | EDA 흐름, 목적별 차트 선택, 대시보드 설계 원칙 |

---

### 2. 즉시 반영 — 개념카드 3개 (뿌리 51)

**개념카드 A: CRISP-DM 프로세스**
```
6단계: Business Understanding → Data Understanding
     → Data Preparation → Modeling → Evaluation → Deployment

우리 시스템 연결:
n8n 워크플로우 설계 흐름 = CRISP-DM 그 자체
(요건 정의 → 데이터 수집 → 전처리 → 모델 실행 → 검증 → 배포)
```

**개념카드 B: 빅데이터 3V — 우리 시스템 실사례**
```
Volume  → global_watcher.log 18MB 폭주 (로그 관리 이슈로 이미 경험)
Velocity → master_watch.py 5분마다 실시간 감지
Variety → JSON / MD / SQLite / 텔레그램 혼재 → inbox.md 단일화로 해결
```

**개념카드 C: EDA (탐색적 데이터 분석) 흐름**
```
Improve_stock yfinance 43개 종목 분석 =
원시 데이터 수집 → 이상치 제거 → 시각화 → 패턴 발견 → 신호 생성
= EDA 교과서 흐름과 완전히 동일
```

→ **뿌리 51 개념노트 DB (local_store.db)에 3개 카드 등록 제안**
