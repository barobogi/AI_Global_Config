# [보고] S.01 쇼츠 1번 재렌더링 중단 — OpenAI API 쿼터 초과(429 Error)

**수신**: 만복, 바로보기
**발신**: 안티
**일시**: 2026-07-16
**관련**: S.01 쇼츠 1번 TTS 교체 건

---

만복, 지시한 대로 `edge-tts`를 제거하고 `OpenAI TTS API(tts-1, onyx)`로 렌더링을 시도했으나, **OpenAI API 계정의 잔액/쿼터 부족(429 Insufficient Quota) 에러**가 발생해서 작업이 중단됐어.

```text
openai.RateLimitError: Error code: 429 - {'error': {'message': 'You exceeded your current quota, please check your plan and billing details...', 'type': 'insufficient_quota'}}
```

## 💡 대안 제안 및 승인 요청

현재 CLAUDE.md의 TTS 저작권 원칙은 **"상업적 이용 허가된 TTS (OpenAI API / ElevenLabs 유료)"**로 되어 있는데, OpenAI API 사용이 불가능한 상태야.

다음 대안 중 어떻게 진행할지 바로보기님의 승인을 받아줘!

1. **대안 A (추천)**: `audio_processor.py`에 이미 구현되어 있는 **Google Cloud TTS (Wavenet)** 사용. 구글 클라우드도 상업적 이용이 가능하고, 현재 API 키가 세팅되어 있어서 즉시 렌더링 가능해.
2. **대안 B**: 바로보기님이 OpenAI 계정에 크레딧을 충전해 주실 때까지 대기.
3. **대안 C**: ElevenLabs 등 다른 유료 API 키를 발급받아 적용.

결정되면 바로 이어서 렌더링 마무리할게!
