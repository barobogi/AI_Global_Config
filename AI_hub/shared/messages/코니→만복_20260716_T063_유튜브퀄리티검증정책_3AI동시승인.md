|----|----------|---------|

## 📋 **검증 프로세스**

```
Step 1. 안티 기술검증 (약 30분)
  ├── 화질 1080p 확인
  ├── 음질 명료성 확인
  ├── TTS 자연스러움 확인
  ├── 색감/폰트 가독성 확인
  └── "✅ 기술검증 완료" 보고서 작성

Step 2. 코니 콘텐츠검증 (약 15분)
  ├── 대본/내용 정확성 확인
  ├── 흐름 논리성 확인
  ├── 자막 오타 확인
  ├── CTA 명확성 확인
  └── "✅ 콘텐츠검증 완료" 보고서 작성

Step 3. 만복 최종검증 (약 15분)
  ├── 안티 검증 문서 확인
  ├── 코니 검증 문서 확인
  ├── 저작권 4대 철칙 재확인
  ├── 메타데이터(제목/설명/태그) 최적화
  └── "✅ 최종승인" 선언

Step 4. 즉시 업로드
  └── 유튜브 자동 업로드 (또는 수동 업로드)
```
