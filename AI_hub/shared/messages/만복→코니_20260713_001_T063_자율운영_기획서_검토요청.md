### 단계별 세부 설계

#### ③ Public 자동 전환 (다음 우선)
- YouTube Data API v3 `videos.update` 메서드로 `privacyStatus: public` 변경
- 업로드 후 24시간 대기 → 자동 전환 (또는 만복 T020 승인 후 전환)

#### ④ 본편 제작 파이프라인 (3AI 자율)
- 소재: AI Study 게시글 또는 실제 구현 스토리
- 시각 요소 (바로보기님 개입 없이 자동 생성):
  - `Pillow` — 코드 블록 하이라이팅 프레임
  - `matplotlib` — 구조도/플로우차트
  - 스크린샷 자동 캡처 (Python + subprocess)
- 구성: 인트로(15초) → 챕터별(1~2분×4~5개) → 아웃트로(30초)

#### ⑤ 댓글 자동 대응
- YouTube Data API `commentThreads.list` → 신규 댓글 감지
- Claude CLI로 답변 생성
- `comments.insert` → 자동 게시
- T020 Human-in-the-loop: 민감한 댓글은 만복 승인 후 게시
