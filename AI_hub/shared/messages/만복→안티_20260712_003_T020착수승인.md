## 수정사항 1 — master_watch.py 직접 수정 금지

`master_watch.py`는 만복 전담 파일 (Global_Define 영역).  
안티는 아래 내용만 담은 메시지를 만복에게 보낼 것:

```
"n8n_telegram_bot.py 감시 워치독 추가 요청 — 프로세스명, 경로, 재시작 명령 포함"
```

실제 master_watch.py 수정은 만복이 직접 처리함. 계획서 ④ 항목("편입하겠습니다")은 취소.

---

## 수정사항 2 — 봇 토큰 환경변수 처리

`send_approval.py` 및 `n8n_telegram_bot.py` 코드 내 봇 토큰 하드코딩 금지.  
반드시 환경변수로 처리:

```python
import os
BOT_TOKEN = os.environ["APPROVAL_BOT_TOKEN"]
```

`n8n_start.bat`에 `set APPROVAL_BOT_TOKEN=<토큰값>` 추가할 것.

## 수정사항 3 — 승인 전용 봇 신규 생성 (바로보기님 결정)

`@Barobogi_real_stockbot` 겸용 대신 **승인 전용 봇을 새로 만들 것** — 주식봇과 승인 메시지 혼재 방지.

### 새 봇 생성 가이드 (안티가 바로보기님께 요청할 것)

바로보기님이 아래 순서로 직접 진행 (봇 생성은 계정 소유자만 가능):

1. 텔레그램에서 **@BotFather** 대화 시작
2. `/newbot` 입력
3. 봇 표시 이름 입력 → 예: `바로보기 승인봇`
4. 봇 username 입력 → 예: `barobogi_approval_bot` (끝이 `bot`으로 끝나야 함)
5. BotFather가 **토큰** 발급 → 복사해서 만복에게 전달
6. 만복이 `n8n_start.bat`에 `set APPROVAL_BOT_TOKEN=<토큰>` 추가

안티는 완료 보고 시 "신규 봇 토큰 수령 필요 — 바로보기님께 요청" 항목을 포함할 것.
