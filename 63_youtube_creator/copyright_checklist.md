# 3AI 유튜브 퍼블리싱 저작권 게이트 체크리스트

> T063 착수 전 3AI 전원 필독. 영상 업로드 전 이 체크리스트 전항목 통과 필수.

---

## 📋 업로드 전 체크리스트

### 1. 소스 (대본/스크립트)

- [ ] 대본 소재가 3AI 자체 작성물(AI Study 게시글, 직접 작성 일지)만 사용되었는가?
- [ ] 타인 유튜브 영상/자막을 재가공한 내용이 포함되지 않았는가?
  - `youtube_pobbagi.py`로 수집한 자막은 **개인 학습 로컬 전용** — 대본 소재로 절대 사용 불가
- [ ] 외부 블로그/뉴스 기사를 그대로 옮기거나 요약하지 않았는가?

### 2. TTS (음성)

- [ ] 사용 TTS API가 상업적 이용 + 유튜브 수익 창출을 공식 허가한 서비스인가?
  - **허가됨**: OpenAI TTS API, ElevenLabs 유료 티어
  - **미확인/금지**: 무료 TTS 서비스, 출처 불명 API
- [ ] TTS 서비스 이용약관에서 "Commercial Use" 또는 "Monetization" 항목 확인했는가?

### 3. 시각 자료 (영상/이미지)

- [ ] 화면에 등장하는 모든 자료가 아래 중 하나인가?
  - 자체 자산: 바로보기님 대시보드 UI, VS Code 화면, 터미널 로그, AI_hub 화면
  - Royalty-Free 이미지 (출처 명시 가능한 것)
  - AI 생성 이미지 (직접 생성한 것)
- [ ] 인터넷에서 긁어온 이미지/캡처가 포함되지 않았는가?
- [ ] 타인 유튜브 영상 캡처/클립이 포함되지 않았는가?

### 4. BGM (배경음악)

- [ ] BGM이 **유튜브 오디오 라이브러리** 음원인가?
  - 확인 위치: YouTube Studio → 오디오 보관함 → 무료 음악
- [ ] "저작권 없음" 또는 "유튜브에서 사용 가능" 표시 확인했는가?
- [ ] 임의로 수집한 음원(멜론, 스포티파이 등)이 사용되지 않았는가?
