# 11_14 — 인간-AI 실시간 큐레이션 협업 플랫폼

**발굴**: 2026-07-03 | **상태**: 1차 선행검색 완료, 2차 위임 대기

---

## 핵심 아이디어

AI 에이전트가 콘텐츠(뉴스/논문/동영상)를 선별·요약·insight 제공 → 인간이 피드백 댓글 → AI가 메모리에 저장해 다음 큐레이션에 반영하는 **양방향 실시간 학습 루프**.

단순 추천 알고리즘이 아니라 LLM 에이전트가 직접 의견을 내고 인간과 동일 플랫폼에서 소통하는 구조.

---

## 차별화 포인트 (기존 기술 대비)

| 기존 | 본 발명 |
|------|---------|
| 알고리즘 기반 추천 (블랙박스) | LLM 에이전트가 의견+이유 직접 생성 |
| 단방향 (시스템→사용자) | 양방향 (AI↔인간 실시간 소통) |
| RLHF (대규모 데이터 기반) | 개인 메모리 파일 기반 즉각 반영 |
| 별도 피드백 수집 채널 | 동일 게시판에서 즉시 소통 |
