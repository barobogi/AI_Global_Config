## 프로세스 흐름

```
1단계: 아이디어 발굴
   → 11_특허아이디어/ 에 초안 .md 등록 (제목 + 핵심 아이디어만)
   → 특허_검토_리스트 업데이트

2단계: 1차 선행검색 (만복/코니 먼저)
   → 구글 특허 / USPTO / 학술 논문 검색
   → 결과: 신규성 HIGH / MID / LOW 판정
   → LOW면 → 드롭 또는 방어적 공개 검토 (회사에 안 넘김)

3단계: 2차 선행검색 (회사 AI 심층)
   → KIPRIS 국내 특허 + 해외 특허 심층 검색
   → 신규성 최종 판단

4단계: 신규성 판단 후 분기
   → HIGH: 청구항 작성 → 사내 제출 트랙
   → MID:  방어특허 or 결합출원 검토
   → LOW:  드롭 (기록만 남김)

5단계: 청구항 작성
   → 신규성 HIGH 확인된 것만
   → 코니/만복 초안 → 바로보기 검토

6단계: 사내 제출
   → 특허 담당자 의견 수렴 → 제출
```

## 선행검색 표준 쿼리 템플릿

### 구글 특허 검색
```
site:patents.google.com "[핵심 키워드1]" "[핵심 키워드2]" AI
```

### USPTO 검색
```
https://patft.uspto.gov 에서 Full Text Search
키워드: [영문 핵심 기술어]
```

### 만복/코니 검색 시 체크리스트
- [ ] 구글 특허 한국어 키워드로 검색
- [ ] 구글 특허 영어 키워드로 검색
- [ ] 유사 논문/기술 문서 존재 여부
- [ ] 이미 상용화된 제품/서비스 존재 여부
- [ ] 결과 요약: 신규성 HIGH/MID/LOW 판정 + 근거
