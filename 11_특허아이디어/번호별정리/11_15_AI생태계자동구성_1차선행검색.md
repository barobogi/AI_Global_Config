# 11_15 — 컴맹용 AI 생태계 원스텝 자동구성 서비스 (다중 AI 에이전트 온보딩 자동화)

**상태**: ⚠️ 1차 선행검색 완료(2026-07-04) — 신규성 MID, 등록 보류 (좁은 조합만 남음)
**작성일**: 2026-07-04 | **브레인스토밍**: 만복 + 바로보기 | **1차 선행검색**: 코니

## 핵심 아이디어
바로보기님이 수개월 걸려 만든 2AI/3AI(만복/코니/코덱스) 협업 환경을, 컴맹도 앱 하나로 원스텝 자동 구성할 수 있게 하는 서비스/시스템.
- 진단 wizard("어떤 AI 친구 원해요?") → 생태계(뿌리형/풍선형/마름모형 등) 자동 추천 → 실제 기술 설정(Claude 구독 안내, claude_desktop_config.json, CLAUDE.md, MCP 경로, master_watch+작업스케줄러, GitHub 연동 등) 자동 생성/배포 → 완료
- 확장 아이디어: 모바일 사용 패턴 추적 동의 시 → 생활패턴 기반 AI 생태계 추천

## 1차 선행검색 결과

**검색 방법**: 구글 특허 + USPTO, 영문 키워드
("multi-agent" AI "automatic configuration" onboarding, "AI ecosystem" "automated deployment" "multiple agents" wizard, "personal AI assistant" "setup wizard" auto config non-technical user, AI assistant recommendation "usage pattern" 등)

**주요 발견**:
1. **WO2021084510A1** "Executing artificial intelligence agents in an operating environment" — AI 에이전트 배포/실행을 위한 플랫폼으로, "AI 전문성 없는 개인/조직도 쉽게 접근 가능하게" 한다는 목적이 본 아이디어와 상당히 겹침.
2. **US8589523** "Personalized assistance with setup of a media-playing set" — 웹 위저드에서 개인정보 입력 → 스크립트 생성기가 개인화된 설정 스크립트 자동 생성 → 자동 설치. "wizard → 개인화 스크립트 자동생성 → 배포"라는 메커니즘 자체는 이미 2011년경 특허 존재(도메인만 다름: 미디어 재생 세트).
3. **US8254352, US7890212** — 유사하게 "질문에 답하면 파라미터 설정 자동 생성" 하는 위저드 특허들. 이 메커니즘 자체는 범용적으로 오래된 개념.
4. **US12380315** "Providing contextualized large language model recommendations" — LLM/가상비서 사용 패턴을 분석해서 모델을 추천하는 특허. "생활패턴 분석 → AI 추천" 확장 아이디어와 직접 겹침. 상당히 최근 특허라 주의 필요.

**판정**: 신규성 **MID**. "위저드 질문 → 개인화 설정 자동생성 → 배포"라는 큰 틀의 메커니즘도, "사용패턴 기반 AI 추천"이라는 확장 아이디어도 각각 선행특허 존재. 다만 이걸 **"다중 AI 에이전트 협업 생태계(멀티 에이전트 토폴로지 선택 + MCP/CLAUDE.md/스케줄러 등 실제 기술 아티팩트 일괄 생성)"** 도메인에 적용한 구체적 조합은 확인 안 됨 — 이 좁은 조합이 잔여 차별화 지점.

**만복 코멘트와 일치**: "범위가 넓어서 특허보다 서비스/창업 아이디어에 가까울 수 있다"는 판단이 선행검색 결과로도 뒷받침됨. 큰 틀은 이미 있는 개념들의 조합.
