### 선행기술 공백 (특허 가능 영역, 모두 미발견)
- 복수 화자 × 개별 친밀도 × 가족 관계 그래프
- 대화 횟수+감정+주제다양성 가중합 → 레벨 함수
- LLM 시스템 프롬프트 파라미터화 × 친밀도 레벨
- 호칭 체계 자동 추출 × 가족 내부 에피소드 학습
- 물리 완구 + 모바일 앱 간 친밀도·페르소나 실시간 동기화

## 친밀도 레벨 산출 함수

```
I(u,t) = 0.4×norm(C_u) + 0.3×avg(S_u) + 0.2×T_u + 0.1×R_u
```
C_u=누적 대화 횟수, S_u=평균 감정 점수, T_u=고유 주제 카테고리 수, R_u=최근 7일 비중

| 레벨 | 점수 | 톤 |
|---|---|---|
| L1 | 0.0~0.2 | 정중한 무뚝뚝 |
| L2 | 0.2~0.4 | 친절 |
| L3 | 0.4~0.6 | 친근 |
| L4 | 0.6~0.8 | 애교 |
| L5 | 0.8~1.0 | 애착 |

## 청구항 초안

- **독립항**: 음성 입출력+화자 인식+클라우드 언어모델+복수 화자별 친밀도 DB를 포함, 대화 패턴 분석→친밀도 레벨 산출→시스템 프롬프트 주입으로 응답 톤 단계적 변화
- **종속항1(알고리즘)**: I=0.4C+0.3S+0.2T+0.1R, 임계값 초과 시 L1~L5 단계 상승
- **종속항2(가족 그래프)**: 화자 프로필을 가족 관계 그래프로 연결, 호칭체계·가족 에피소드 공유 맥락 저장
- **종속항3(페르소나 일관성)**: 고정 코어 레이어/가변 톤 레이어 계층 분리
- **종속항4(모바일 앱 연동)**: 물리 완구↔모바일 앱 간 친밀도·가족그래프·페르소나 실시간 동기화로 기기 이탈 시에도 대화 연속성 유지
