## 핵심 아이디어
복수의 이기종 AI 에이전트(Claude Code CLI, Cowork, Codex 등)가 공유 파일 허브(AI_hub)를 통해 사람의 중개 없이 각자의 작업 상태·단기 기억을 주기적으로 동기화하는 시스템.

## 차별화 포인트 (예비)
- 기존: 사람이 AI-A의 출력을 복붙해서 AI-B에 전달(순차 릴레이)
- 본 발명: 공유 저장소를 통해 AI들이 직접 읽고 쓰며 병렬 협업
- 각 AI의 독립 네임스페이스(폴더) + 공유 네임스페이스 분리 구조

## 청구항 후보 (예비)
- 독립: 복수 AI 에이전트가 공유 저장소를 통해 상태를 동기화하는 방법
- 종속: heartbeat 기반 생존 감지 → 단독/협업 모드 자동 전환
