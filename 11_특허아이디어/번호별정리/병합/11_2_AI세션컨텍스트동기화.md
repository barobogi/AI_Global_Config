## 발명의 명칭
AI 에이전트 세션 간 자동 컨텍스트 동기화 시스템 (Automatic Context Synchronization System Between AI Agent Sessions)

## 기술 분야
AI 어시스턴트 세션이 종료되거나 전환될 때 파일 시스템 감시를 통해 다음 AI 에이전트가 판독 가능한 컨텍스트 파일을 자동 생성·동기화하는 시스템.

## 발명의 배경
AI 어시스턴트는 세션 종료 시 컨텍스트가 소멸되며, 다음 세션/다른 에이전트가 이어받으려면 사용자가 수동으로 내용을 전달해야 함 → 반복적 맥락 재설명, 작업 연속성 저하.

## 발명의 목적
1. 세션 종료/전환 시 컨텍스트 자동 보존
2. 복수 이기종 AI 에이전트 간 표준화된 컨텍스트 전달
3. 사용자 개입 없는 완전 자동화된 세션 브릿지

## 핵심 구성요소
파일 시스템 감시 모듈(watchdog) → 변경 감지 → 컨텍스트 추출 모듈(현재 세션 요약) → 구조화된 브릿지 파일 자동 생성(Markdown/JSON) → 다음 AI 에이전트 세션 시작 시 자동 로드

## 신규성 및 차별점

| 구분 | 기존 기술 | 본 발명 |
|---|---|---|
| 컨텍스트 보존 | 세션 종료 시 소멸 | 파일로 자동 영속화 |
| 전달 방식 | 사용자 수동 복사붙여넣기 | 감시 기반 자동 생성 |
| 대상 | 단일 AI 플랫폼 내부 | 이기종 AI 에이전트 간 |
| 플랫폼 | 특정 서비스 종속 | 파일 시스템 기반 독립 |

## 핵심 청구항 후보
파일 시스템 감시 모듈이 AI 세션 종료 이벤트를 감지하여 다음 AI 에이전트 세션이 판독 가능한 구조화된 컨텍스트 파일을 자동 생성하는 방법 및 시스템

## 특허 가능성 (원 평가)
중간 — 순수 SW 발명으로 단독 특허는 어려우나, 파일 감시+자동생성+AI 에이전트 연동 결합 시스템 청구항으로 가능성 있음.

## 왜 제외되었나
실제 구현(만복1↔코니 동기화 시스템, master_watch.py 자동 브릿지 생성)이 특허3의 "Layer A 개별 에이전트 상태 계층 + 온보딩 브리핑" 구조와 완전히 겹침. 독립 출원보다 특허3 종속항1로 흡수하는 것이 더 강한 청구항 구성(이기종 생존판정+협업모드 자동전환까지 포함)이라 판단.
