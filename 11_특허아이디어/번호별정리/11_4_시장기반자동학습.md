## 발명의 명칭
인간 판단 배제 시장 결과 기반 투자 신호 자동 최적화 시스템 (Human Bias-Free Automated Investment Signal Optimization System Using Market Outcome Feedback)

## 기술 분야
투자 신호 생성 후 실제 시장 가격 변동을 자동 수집하여, 인간 판단 개입 없이 신호 패턴별 가중치를 자동 조정하는 투자 신호 최적화 시스템.

## 발명의 배경
기존 알고리즘 트레이딩 최적화는 강화학습(대규모 인프라 필요), 전문가 피드백(인간 편향 개입), 백테스팅(사후 검증만 가능) 방식이 주. 특히 인간 피드백 기반 학습은 투자자 개인의 심리적 편향이 시스템에 학습되는 문제가 있음.

## 발명의 목적
1. 시장 가격만을 유일한 학습 신호로 사용 → 인간 편향 완전 배제
2. 개인 투자자 수준에서 구현 가능한 경량 자동학습 파이프라인
3. 단순 명확한 피드백 기준(N일 후 수익률)으로 복잡한 보상 함수 불필요

## 핵심 구성요소
투자 신호 생성 모듈(AI 등급 산출) → 신호+날짜 저장 → 시장 데이터 자동 수집(N일 후) → 패턴별 실제 수익률 계산 → 신호 가중치 자동 조정(인간 개입 없음) → 다음 신호 생성에 반영

## 신규성 및 차별점

| 구분 | 기존 기술 | 본 발명 |
|---|---|---|
| 학습 신호 | 전문가 판단 또는 복잡한 보상함수 | 시장 가격(객관적 데이터)만 사용 |
| 인간 편향 | 피드백 과정에서 개입 | 완전 배제 |
| 인프라 | 대규모 RL 시스템 필요 | 경량 파이프라인(개인 운용 가능) |
| 적용 대상 | 기관 투자자 | 개인 투자자 |

## 핵심 청구항 후보
투자 신호 생성 후 일정 기간 경과 시 시장 가격 데이터를 자동 수집하여 인간 판단 개입 없이 신호 패턴별 가중치를 갱신하는 자동화된 투자 신호 최적화 방법 및 시스템
